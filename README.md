# ASPNET_Core_DDD_4
Aplicação em ASP.Net Core 3.1 (Blazor). Arquitetura em quatro camadas (application/domain/service/infrastructure) baseada nos princípios DDD.

## Recursos utilizados:

- DDD (Domain-Driven Design);
- Visual Studio 2019;
- ASP.Net Core 3.1;
- Entity Framework Core;
- Blazor;
- Bootstrap;
- Sass.
